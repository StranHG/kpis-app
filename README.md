# DiDi Food Oaxaca — Dashboard de KPIs

Panel de control gerencial para DiDi Food en Oaxaca. Muestra KPIs operativos, análisis por área y genera reportes PDF.

## Stack

- React Native + Expo (TypeScript)
- Node.js + Express
- PostgreSQL (Supabase)

## Correr el proyecto

**API**

```bash
cd api
npm install
node index.js
```

Requiere `api/.env` con `DATABASE_URL=postgresql://...`

**App**

```bash
npm install
npx expo start
```
